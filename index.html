Apa itu Algoritma?

Algoritma adalah urutan langkah-langkah logis untuk menyelesaikan masalah.

Karakteristik Algoritma:

  • Input: Memiliki nol atau lebih masukan
  • Output: Memiliki minimal satu keluaran
  • Definiteness: Setiap langkah jelas
  • Finiteness: Harus berakhir
  • Effectiveness: Dapat dikerjakan

Cara Penulisan Algoritma

1. Deskriptif

Algoritma Luas Persegi Panjang: 1. Mulai 2. Masukkan panjang 3. Masukkan lebar 4. Hitung luas = panjang √ó lebar 5. Tampilkan luas 6. Selesai

2. Pseudocode

MULAI INPUT panjang, lebar luas ‚Üê panjang * lebar OUTPUT luas SELESAI

Struktur Dasar Pemrograman

a) Sequence (Berurutan)

Instruksi dijalankan berurutan dari atas ke bawah

b) Selection (Percabangan)

IF nilai >= 75 THEN OUTPUT "Lulus" ELSE OUTPUT "Tidak Lulus"

c) Iteration (Perulangan)

FOR i = 1 TO 10 DO OUTPUT i END FOR

üìù Contoh Soal

Contoh: Menentukan Bilangan Genap/Ganjil

Algoritma: 1. Mulai 2. Input bilangan 3. Jika bilangan mod 2 = 0 Tampilkan "Genap" Jika tidak Tampilkan "Ganjil" 4. Selesai

Contoh: Mencari Nilai Terbesar dari 3 Bilangan

Algoritma: 1. Mulai 2. Input A, B, C 3. Maks = A 4. Jika B > Maks maka Maks = B 5. Jika C > Maks maka Maks = C 6. Tampilkan Maks 7. Selesai
